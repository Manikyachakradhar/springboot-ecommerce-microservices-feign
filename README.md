# Spring Boot E-Commerce Microservices (OpenFeign)

Microservices-based e-commerce backend built using:

- Spring Boot 3.5
- OpenFeign
- Spring Data JPA
- PostgreSQL
- Maven

Services:
- Product Service
- Cart Service
- Order Service

Features:
- Product Management
- Cart Management
- Order Creation
- Checkout Flow
- Stock Validation

Upcoming:
- API Gateway
- Eureka Service Discovery
- Docker
